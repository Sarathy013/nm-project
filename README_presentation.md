# Notebooks and Scripts

## Primary Notebook (primary_notebook.ipynb)

Demonstrates the Medical Document Intelligence System by ingesting data, running queries, and evaluating performance.

### Prerequisites
- Backend server running on localhost:5000
- Ollama with LLaMA 3 installed

### Usage
1. Install dependencies: `pip install -r requirements.txt`
2. Start the backend: `cd backend && npm run dev`
3. Run the notebook cells.

## Presentation Deck Generator (presentation_deck.ipynb)

Generates a PowerPoint presentation for the project.

### Usage
1. Install dependencies: `pip install -r requirements.txt`
2. Run the notebook to generate `Medical_Document_Intelligence_Presentation.pptx`

## Troubleshooting
- Ensure the backend is running for the primary notebook.
- For presentation, python-pptx requires additional system dependencies on some platforms.
